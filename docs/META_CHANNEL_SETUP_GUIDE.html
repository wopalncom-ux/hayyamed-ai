STEP 1 · Create your Meta foundation YOU · ~20 min

  1. Go to business.facebook.com → create a Meta Business Account (if you don't have one).
  2. Add your Facebook Page and Instagram Business account to the Business Account (Instagram must be a Business/Professional account, linked to the Page).
  3. Go to developers.facebook.com → My Apps → Create App → type "Business" → name it e.g. Hayya AI CRM.
  4. In the app, note your App ID and App Secret (Settings → Basic).
⚠️ Meta requires App Review before these work for real customers (not just test numbers). You can build & test in Development mode immediately; submit for review when ready to go live. Permissions to request: whatsapp_business_messaging, instagram_manage_messages, pages_messaging.

STEP 2 · WhatsApp Business (Cloud API) Platform ready now

Your platform already has a working WhatsApp webhook — this is the fastest channel to light up.

2a. In the Meta app

  1. Add the WhatsApp product to your app.
  2. Under WhatsApp → API Setup, you'll get a test number + a temporary token. For production, add your real business number and generate a permanent access token (via a System User in Business Settings).
  3. Copy three things: the Phone Number ID, the permanent access token, and your App Secret (from Step 1).

2b. Configure the webhook (in Meta → WhatsApp → Configuration)

Callback URL: https://api.hayyaai.com/api/v1/whatsapp/webhook
Verify token: a phrase you invent, e.g. hayya-verify-2026 — you'll set the same value in Secret Manager (Step 2c)
Subscribe to: the messages field

2c. Put the keys in GCP Secret Manager YOU

Go to Secret Manager (project hayyamed-ai-crm) → for each secret, click it → + New Version → paste → Add:

SecretValue to paste
hayyamed-whatsapp-tokenYour permanent WhatsApp access token
hayyamed-whatsapp-webhook-tokenThe verify token you invented (must match Meta exactly)
hayyamed-meta-app-secretYour Meta App Secret
Then tell me "WhatsApp keys are in" — I roll a new API revision and connect your number in /integrations/whatsapp (enter the Phone Number ID). We send a test message to confirm 2-way flow.

STEP 3 · Instagram DMs Build step + YOU

3a. In the Meta app YOU

  1. Add the Instagram product (Instagram Graph API / Messaging).
  2. Connect your Instagram Business account (linked to your Page).
  3. Generate a Page access token with Instagram messaging permissions.
  4. In Webhooks, subscribe the app to instagram → field messages.

3b. What I build

The platform has the Instagram channel model + integrations UI, but the inbound webhook handler for Instagram isn't wired yet. Once your app + token exist, I add an /api/v1/instagram/webhook handler (mirrors the WhatsApp one) so IG DMs flow into the same inbox with AI replies. ~1 focused build + deploy.

STEP 4 · Facebook Messenger Build step + YOU

4a. In the Meta app YOU

  1. Add the Messenger product.
  2. Generate a Page access token for your Facebook Page.
  3. In Messenger → Webhooks, subscribe to messages and messaging_postbacks.

4b. What I build

Same as Instagram — I add a Messenger webhook handler so Page messages land in your inbox. (Instagram + Messenger share Meta's Graph webhook, so I build them together.)
